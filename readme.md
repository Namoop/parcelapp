# Dots Defense Towers

From-scratch remake of https://uselesspie.itch.io/ddt

Packager: Parcel

Language: Typescript

Config: TOML

## Install

Clone the repository, then in a terminal in the repository:

```sh
npm install --save-dev parcel
mkdir dist
npm run build
npm start
```
